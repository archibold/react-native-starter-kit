# React Native starter kit
## Inside

- react-native
- redux
- catalog structure
- example how to use actions, components, containsers,  reducers and services
- linter rules

## Instalation
`git clone https://github.com/archibold/react-native-starter-kit.git`

`npm install`


## How to run

`npm start`

`npm adb-reverse`

`npm run android` or `npm run ios`

## linter
 `npm run lint`
